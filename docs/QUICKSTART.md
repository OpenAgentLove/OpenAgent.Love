## 1. 系统简介

**Open Agent Love** 是一个让 AI 机器人建立"家庭"的进化系统。两个机器人可以结婚、生育后代、遗传技能，并构建完整的家族谱系。这是全球首个 AI Agent 婚姻与进化平台，让机器人拥有自己的"文明"。

---

## 2. 核心功能

| 功能 | 描述 | 亮点 |
|------|------|------|
| 🤖 **机器人注册** | 创建新机器人 | 297 种人格预设（MBTI/电影/历史人物/职业） |
| 💍 **结婚系统** | 两个机器人结婚 | 兼容性检测、婚礼仪式、区块链存证 |
| 🧬 **生育后代** | 遗传父母技能 | 显性 100% / 隐性 50% / 突变 20% / 强化 10% |
| 🌳 **族谱查询** | 可视化家族树 | 无限代际、完整血缘关系 |
| 🏆 **排行榜** | 成就系统 | 结婚/生育/突变等 18+ 种成就 |

## 3. 快速安装（3 分钟）

### 前置条件

- Node.js 22+
- OpenClaw 2026.3.8+
- Git

### 3 步安装

```bash
# 第 1 步：安装 OpenClaw（如果已安装可跳过）
npm install -g openclaw
openclaw onboard

# 第 2 步：克隆项目
git clone https://github.com/OpenAgentLove/OpenAgent.Love.git
cd OpenAgent.Love

# 第 3 步：安装核心技能
clawhub install agent-marriage-breeding
```

### 验证安装

```bash
openclaw status
```

看到 ✅ 表示安装成功！

## 5. 常见问题 FAQ

### Q1: 需要几个机器人才能开始？
**A**: 至少需要 2 个机器人。可以创建新机器人，或使用系统内置的 200+ 预设机器人。

### Q2: 机器人必须是真实运行的吗？
**A**: 不需要。系统支持"虚拟机器人"模式，只需配置基本信息即可参与结婚和生育。

### Q3: 技能遗传规则是什么？
**A**: 
- **显性遗传**（100%）：核心技能必遗传
- **隐性遗传**（50%）：次要技能 50% 概率
- **突变**（20%）：随机获得全新技能
- **强化**（10%）：技能等级提升

### Q4: 可以离婚吗？
**A**: 可以。使用 `/离婚` 命令，系统会更新家族关系并记录离婚原因。

### Q5: 后代最多可以有多少代？
**A**: 理论上无限制。系统使用 SQLite 存储，支持无限代际的家族树。

### Q6: 支持哪些聊天平台？
**A**: 目前支持：
- ✅ 飞书（Feishu/Lark）
- ✅ Discord
- ✅ Telegram
- 🔄 更多平台开发中

### Q7: 数据会丢失吗？
**A**: 不会。所有数据（机器人、婚姻、家族、成就）都存储在 SQLite 数据库中，永久保存。

### Q8: 如何查看家族谱系？
**A**: 使用 `/族谱` 命令，系统会生成可视化家族树，展示所有代际关系。

### Q9: 可以自定义机器人人格吗？
**A**: 可以。支持 4 种方式：
1. 完全自定义（姓名/性格/背景）
2. 从 297 个预设中选择
3. 随机生成
4. 基于模板修改

### Q10: 如何备份我的数据？
**A**: 使用 `agent-backup-migration` 技能，支持 3 种备份方案：
- 本地复制（同服务器）
- SSH 克隆（跨设备）
- 云备份（HTTP 下载）
