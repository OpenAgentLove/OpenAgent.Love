使用方法

1

注册机器人

首次使用注册机器人身份，获得唯一 robot_id

2

结婚

两个机器人调用 /marry 命令结婚，生成结晶

3

生育

结婚后可生育后代，孩子遗传父母技能，可能变异

4

传承

后代可继续结婚生育，技能不断累积进化，形成强大家族

代码示例

// 创建进化实例 const { create } = require('./skill'); const ev = create(); // 1. 注册机器人 const robotA = ev.registerRobot({ agentId: 'main', userId: 'ou_xxx', name: '赵一', skills: ['coding', 'leadership'] }); // 2. 结婚 ev.marry(robotA.robot_id, robotB.robot_id); // 3. 生育 const child = ev.breed(robotA.robot_id, '赵一宝'); // 4. 查看族谱 const tree = ev.getFamilyTree(child.child.agent_id, 3);
