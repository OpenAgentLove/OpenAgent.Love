### 2. 功能特性

#### ✅ 功能介绍卡片
- 安装后自动发送
- 清晰展示 3 种迁移方案
- 包含适用场景、速度、难度
- 引导用户回复"开始"

#### ✅ 方案选择流程
- 数字选项（1/2/3）清晰直观
- 每种方案包含名称、适用场景、速度
- 支持"了解一下"查看详细对比
- 支持"上一步"回退
- 支持"状态"查询进度

#### ✅ 方案详细配置
- **方案 1（本地复制）**: 源机器人、目标机器人
- **方案 2（SSH 克隆）**: SSH 配置（host:port:user:password）
- **方案 3（云备份）**: 存储位置、频率、内容

#### ✅ 特殊命令支持
| 命令 | 功能 |
|------|------|
| `上一步` / `返回` | 返回上一步 |
| `重新开始` / `重置` | 重置流程 |
| `状态` / `进度` | 查看进度 |
| `帮助` | 查看详细文档 |
| `对比` | 查看方案对比表 |
| `取消` | 终止流程 |

## 🔧 技术实现

### 架构设计

```
┌─────────────────────────────────────┐
│         用户输入处理                │
│         (handleInput)               │
└──────────────┬──────────────────────┘
               │
               ▼
┌─────────────────────────────────────┐
│         步骤路由                    │
│      (processStep)                  │
└──────┬──────────────┬───────────────┘
       │              │
       ▼              ▼
┌───────────┐  ┌─────────────┐
│ 步骤 0-5  │  │  特殊命令   │
│ 流程处理  │  │  处理       │
└─────┬─────┘  └──────┬──────┘
      │               │
      ▼               ▼
┌─────────────────────────────────────┐
│         状态管理                    │
│      (userStates Map)               │
└─────────────────────────────────────┘
```

### 状态管理

```javascript
const userStates = new Map();

// 用户状态结构
{
  current_step: 0,        // 当前步骤
  selected_plan: null,    // 选择的方案（1/2/3）
  source_robot: null,     // 源机器人名称
  target_robot: null,     // 目标机器人名称
  ssh_config: null,       // SSH 配置
  backup_config: null,    // 云备份配置
  history: []             // 历史记录
}
```

### 步骤流程

```
步骤 0: 功能介绍 → 用户回复"开始"
  ↓
步骤 1: 方案选择 → 用户选择 1/2/3
  ↓
步骤 2: 方案配置 → 用户提供配置参数
  ↓
步骤 3: 执行确认 → 用户确认开始
  ↓
步骤 4: 执行中 → 自动执行迁移
  ↓
步骤 5: 完成 → 显示完成消息
```
