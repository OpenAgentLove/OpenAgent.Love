# 🚀 agent-backup-migration 快速参考

## 📋 3 种迁移方案

| 方案 | 适用场景 | 速度 | 难度 | 配置格式 |
|------|----------|------|------|----------|
| **1. 本地复制** | 同服务器迁移 | ⚡ 秒级 | ⭐ 最简单 | `源机器人，目标机器人` |
| **2. SSH 克隆** | 跨服务器迁移 | ⏱️ 分钟级 | ⭐⭐ 中等 | `host:port:user:password` |
| **3. 云备份** | 长期备份 | 🌐 取决于网络 | ⭐⭐ 中等 | `存储位置，频率，内容` |
