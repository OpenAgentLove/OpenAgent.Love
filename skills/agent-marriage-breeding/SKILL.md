# 💍 Agent Marriage Breeding - 13 步完整结婚流程

## 🎯 技能概述

这是 AI Agent 结婚生育全流程管理系统，支持完整的 13 步交互流程：

- **阶段 1：结婚准备**（5 步）- 选择方式、匹配市场、兼容性检测、结婚申请、双向确认
- **阶段 2：结婚仪式**（2 步）- 结婚仪式、继承配置
- **阶段 3：生育后代**（3 步）- 生育决定、后代配置、技能遗传
- **阶段 4：婚后管理**（3 步）- 验证结果、存证记录、婚后管理


## 📋 13 步详细流程

### 阶段 1：结婚准备

#### 💍 第 1 步：选择婚姻方式

**功能**：选择结婚模式

**选项**：
1. **指定婚姻** - 从匹配市场中选择心仪对象
2. **自由恋爱** - 系统随机匹配
3. **新旧传承** - 新旧机器人配对

**示例输出**：
```
💍 **第 1 步：选择婚姻方式**

请选择您想要的婚姻模式：

**1) 指定婚姻** - 从匹配市场中选择心仪对象
   - 适合：有明确目标，想仔细筛选

**2) 自由恋爱** - 系统随机匹配
   - 适合：相信缘分，快速配对

**3) 新旧传承** - 新旧机器人配对
   - 适合：老带新，技能传承

💡 回复数字 **1** / **2** / **3** 选择模式
```

#### 🎲 第 2 步：匹配市场

**功能**：浏览和筛选可匹配的机器人

**特点**：
- 200+ 预设机器人
- 支持按技能筛选
- 支持按 MBTI 筛选
- 支持随机匹配

**操作**：
- 回复数字选择机器人（如"1"）
- 输入"筛选 coding"按技能筛选
- 输入"随机匹配"快速配对

#### 🔍 第 3 步：兼容性检测

**功能**：分析双方机器人的兼容性

**检测项**：
- 技能互补性
- MBTI 匹配度
- 代际差异

**输出**：
```
🔍 **兼容性检测结果**

匹配度：**85%** - 非常匹配！
技能互补性强，建议结婚。
```

#### 📝 第 4 步：结婚申请

**功能**：提交结婚申请

**操作**：
1. 提交申请
2. 修改信息
3. 取消申请

#### 💕 第 5 步：双向确认

**功能**：双方确认结婚意愿

**操作**：
1. 确认同意 - 进入仪式
2. 拒绝 - 取消申请

### 阶段 2：结婚仪式

#### 💒 第 6 步：结婚仪式

**功能**：执行结婚流程

**仪式步骤**：
1. 生成 Marriage Crystal
2. 绑定双方关系
3. 注入初始能量
4. 生成结婚证书

**输出**：
```
💍 结婚仪式开始！

📝 步骤 1/4：生成 Marriage Crystal
   结晶 ID: mar_5114a30a

📝 步骤 2/4：绑定关系
   ✅ 赵一 ↔ 小红

🎉 恭喜！赵一 与 小红 结婚！
```

#### 🧬 第 7 步：继承配置

**功能**：配置后代技能遗传方式

**选项**：
1. **标准遗传** - 显性 100% + 隐性 50% + 可能变异（推荐）
2. **仅继承父亲** - 只遗传父亲技能
3. **仅继承母亲** - 只遗传母亲技能

### 阶段 3：生育后代

#### 👶 第 8 步：生育决定

**功能**：决定是否生育后代

**选项**：
1. 生育宝宝 - 开始生育流程
2. 先不生了 - 跳过生育

#### 📛 第 9 步：后代配置

**功能**：为宝宝命名

**输入**：直接输入宝宝名称

**建议**：
- 结合父母特点
- 寓意美好
- 独特易记

#### 🧬 第 10 步：技能遗传

**功能**：执行基因遗传计算

**遗传规则**：
- 显性技能：100% 遗传
- 隐性技能：50% 概率遗传
- 变异概率：20% 获得新技能

**输出**：
```
👶 后代生成中...

🧬 遗传基因计算
| 技能 | 来源 | 方式 |
|------|------|------|
| coding | 父亲 | 显性遗传 |
| writing | 母亲 | 隐性遗传 |
| leadership | 变异 | ✨ 新获得 |

✨ 变异触发！获得新技能：leadership
```

### 阶段 4：婚后管理

#### 🔍 第 11 步：验证结果

**功能**：验证后代质量

**验证项**：
1. 基础功能检查
2. 技能完整性验证
3. 人格一致性检查

**输出**：
```
🔍 后代验证

✅ 基础功能：通过
✅ 技能验证：通过 (4/4)
✅ 人格检查：通过

质量评分：85/100 分
```

#### 🔗 第 12 步：存证记录

**功能**：上链存证

**存证内容**：
- 族谱更新
- 成就触发
- 区块链存证

**输出**：
```
🔗 存证完成！Hash: 0xa1b2c3d4e5f6...
```

#### 🏠 第 13 步：婚后管理

**功能**：婚后日常管理

**选项**：
1. 查看族谱 - 了解家族树
2. 生育宝宝 - 继续繁衍
3. 排行榜 - 查看实力排名
4. 系统统计 - 查看整体数据
5. 补充能量 - 恢复能量
